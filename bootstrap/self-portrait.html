Code As Art



P5.js Self Portrait


With this project we learn what the layout of the p5.js canvas and how to put objects 
directly into it, we adopt the age old idea of finding a way to creatively represent 
our image and create a coded image that we can use for future branding. We will share the code 
that we create and critique and offer feedback and help to our peers. It is important as 
beginners to connect the code with the action and to be able to share code with each other.
